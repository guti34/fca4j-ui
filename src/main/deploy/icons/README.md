# Icônes FCA4J UI

Ce dossier doit contenir les icônes de l'application pour chaque plateforme.

## Fichiers requis

| Fichier         | Plateforme | Format | Taille recommandée |
|-----------------|------------|--------|-------------------|
| `fca4j-ui.ico`  | Windows    | ICO    | 256x256 minimum   |
| `fca4j-ui.png`  | Linux      | PNG    | 512x512           |
| `fca4j-ui.icns` | macOS      | ICNS   | Multi-résolution  |

Le fichier `fca4j-ui.png` fourni est un placeholder — remplacez-le par
l'icône officielle du projet avant de distribuer.

Voir `README-deploy.md` à la racine du projet pour les instructions
de conversion depuis un PNG source.
